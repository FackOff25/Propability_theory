Программа для задания №3 типового расчёта №1 по дисциплине "Теория вероятности и математичуская статистика" студента МГТУ им. Баумана группы РК6-33Б Козлова Владимира.
Программа написана на языке Java. JDK 12.
Программа включает класс для общих расчётов вероятностей для трёх урн с шариками трёх цветов, остальное вынесено в основной файл.
Программа позволяет вводить любые значения для вычислений, но для ускорения добавлен блок с заранее введёнными значениями для моего варианта (3 группа 6 человек по списку).
Программа на github: https://github.com/FackOff25/Propability_theory_type_1
